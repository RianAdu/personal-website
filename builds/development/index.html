Hi there!
I’m Florian Adu-Gyamfi, a passionate Front End Web Developer with 5+ years of professional experience, based in Toronto, Canada.

My favourite project recipe is a cup of HTML, CSS and JavaScript with a tablespoon of jQuery, adding a teaspoon of AJAX, a pinch of JSON and a couple of Sass partials to it. Then mix it all up with Bootstrap and run it through Gulp to craft high quality responsive websites and to create exceptional user experience.
