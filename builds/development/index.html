about me

Hi there!
I’m Florian, a passionate Front End Developer with 5+ years of professional experience, based in Toronto, Canada.

Born and raised in Germany with a Bachelor's Degree in Media Computer Science, I use web technologies to turn your creative ideas into reality. I enjoy writing code to build beautiful websites and responsive emails. While focusing on intuitive user experience, innovative designs and the latest web development best practices, I always make sure to follow industry standards and to create websites that are easily maintainable and can be enhanced continuously.

My favourite recipe for a web project is a cup of HTML, CSS and JavaScript with a tablespoon of jQuery, adding a teaspoon of AJAX, a pinch of JSON and a couple of Sass partials to it. I either mix this all up with Bootstrap and run it through some Gulp tasks or I use WordPress and implement customized plugins. You don’t know what all this means? Don’t worry, I got you covered! The result is a high-quality responsive website with exceptional user experience.
